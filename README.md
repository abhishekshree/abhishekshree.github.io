# abhishekshree.github.io

Personal portfolio and blog built with [Next.js](https://nextjs.org/) (static export), [Tailwind CSS](https://tailwindcss.com/), and [MDX](https://mdxjs.com/). Deployed via GitHub Actions to [abhishekshree.github.io](https://abhishekshree.github.io).

## Development

```bash
nvm use
npm install
npm run dev
```
